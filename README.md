# Codewars Solutions Repository

This repository holds my solutions to kata (programming challenges) hosted at [codewars.com](https://www.codewars.com). Solutions are sorted by language, then tier, then by the unique id in the problem's link. For example, ./js/kyu8/568018a64f35f0c613000054 provides a solution for the problem at [codewars.com/kata/568018a64f35f0c613000054/](https://www.codewars.com/kata/568018a64f35f0c613000054/).

## "Academic" Integrity
Please do not take my solutions and submit them as your own unaltered.

## Nix?
While there is a nix flake provided for devshell purposes, it is not required to run any of my code.
